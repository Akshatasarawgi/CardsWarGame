# Cards War Game 
Cards War Game is a simple cards game , that uses deckofcardsapi to fetch a new deck of cards and draw two cards each time to check which one of the 
cards has the highest value.

## Table of Contents

- [Overview](#overview)
- [Screenshot](#screenshot)

## Overview
- This is a practice project to gain stronger understanding of fetch api and Promise. To be able to get a new deck of cards 
and draw two cards on every click. The card with the highest value gets a score.
- I used Vanilla Java Script to manipulate DOM based on which card scores , and display the winner of the game at the end. 
- Responsive design with mobile first approach.
